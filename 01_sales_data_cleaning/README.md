# Sales Data Cleaning Demo

## Problem
The dataset contains duplicate rows, inconsistent product names, and missing values in the quantity column.

## Solution
A Python automation script was used to clean the dataset automatically.

## Cleaning Actions Performed
- Removed duplicate rows
- Standardized product name formatting
- Detected missing values
- Exported a clean Excel file

## Files Included
- raw_data.xlsx → original messy dataset
- cleaned_data.xlsx → cleaned output
- clean_script.py → Python automation script
- before.png → screenshot of messy dataset
- after.png → screenshot of cleaned dataset
