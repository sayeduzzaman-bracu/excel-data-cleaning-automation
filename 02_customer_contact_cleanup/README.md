# Customer Contact List Cleanup

## Problem
The dataset contains duplicate customer entries, inconsistent email capitalization, extra spaces in names, and missing phone numbers.

## Solution
A Python script was used to automatically clean and standardize the contact list.

## Cleaning Actions Performed
- Removed duplicate contacts
- Standardized text formatting
- Trimmed extra spaces
- Detected missing phone numbers

## Files Included
- raw_data.xlsx → original messy contact list
- cleaned_data.xlsx → cleaned contact dataset
- clean_script.py → Python cleaning automation
- before.png → screenshot of messy data
- after.png → screenshot of cleaned result
