# Inventory Dataset Cleaning

## Problem
The inventory dataset contains duplicate products, inconsistent product naming, and missing stock values.

## Solution
A Python automation script was used to clean and standardize the inventory data.

## Cleaning Actions Performed
- Removed duplicate product entries
- Standardized product names
- Detected missing stock values
- Exported cleaned dataset

## Files Included
- raw_data.xlsx → original messy inventory data
- cleaned_data.xlsx → cleaned dataset
- clean_script.py → Python automation script
- before.png → screenshot of messy inventory data
- after.png → screenshot of cleaned results
